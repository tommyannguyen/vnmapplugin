# VnMapPlugin Docs Refactor Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all `@rnmapbox/maps` / Mapbox branding across all `.md` files, add prop-level example links to core component docs, and relocate internal working notes.

**Architecture:** Fix the EJS template first so `yarn generate` produces clean output permanently. Then sweep manually written and root docs. Finally add prop-level example tags to 6 core example files and regenerate.

**Tech Stack:** Node.js, EJS (template engine via `ejs` package), `yarn generate` pipeline, `git mv` for file relocation.
